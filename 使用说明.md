# 苏州市电化教育馆官网爬取资料使用说明

生成日期：2026-05-15

## 已生成内容

- `articles/`：逐篇文章 Markdown，共 999 篇。
- `articles_index.csv`：文章总索引，含 ID、标题、栏目、正文长度、附件数量、原文 URL、本地 Markdown 路径。
- `articles.json`：结构化全文数据，便于后续检索和二次整理。
- `site_nav.json`：官网栏目与入口链接。
- `list_pages.json`：栏目列表页抓取记录。
- `errors.json`：抓取错误记录，当前为空。
- `面试复习索引.md`：按 021 工程师岗位相关主题整理的优先复习清单。

## 面试复习优先级

建议先看 `面试复习索引.md`，再按主题打开具体文章。优先顺序：

1. 人工智能与智慧校园
2. 教育数字化平台
3. 网络与数据安全
4. 中心机房与城域网
5. 教师培训与竞赛
6. 教育 E 卡通

## 使用口径

021 岗位应按“计算机工程师 + 教育数字化项目人员 + 人工智能教育应用落地人员”准备。复习时不要只背技术术语，要把官网材料转化为以下表达：

- 技术服务教育治理
- 平台服务师生应用
- 数据服务个性化学习和精准管理
- 人工智能应用必须安全、可控、可解释
- 网络安全和数据安全是教育数字化底座
